# 生成菜单

## 安装插件
使用插件 [vite-plugin-vitepress-auto-sidebar](https://github.com/QC2168/vite-plugin-vitepress-auto-sidebar) 实现


使用方法：
```ts
import AutoSidebar from 'vite-plugin-vitepress-auto-sidebar';

export default defineConfig({
   vite: {
    plugins: [
      AutoSidebar({
        path: '',
      })
    ],
  },
})
```
